# Homebrew Tap for Par Particle Life

Homebrew tap for [Par Particle Life](https://github.com/paulrobello/par-particle-life) - GPU-accelerated particle life simulation in Rust.

## Installation

```bash
brew install paulrobello/par-particle-life/par-particle-life
```

Or tap first, then install:

```bash
brew tap paulrobello/par-particle-life
brew install par-particle-life
```

## About

Par Particle Life is a GPU-accelerated particle life simulation built with Rust and wgpu. Particles of different types attract and repel based on interaction matrices, creating emergent life-like behaviors.
